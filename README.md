# Fix syntax highlight

A syntax highlight for [Fix lang](https://github.com/tttmmmyyyy/fixlang) in VSCode.

This extension colors Fix code with a static **TextMate grammar**, so it works on its own without the `fix` compiler. If you use the [Fix language client](https://marketplace.visualstudio.com/items?itemName=tttmmmyyyy.fixlang-language-client) extension, you do **not** need this one — the language client provides syntax coloring through **semantic tokens** (which requires the `fix` compiler).
